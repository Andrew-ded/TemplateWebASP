# TemplateWebASP
Шаблон для проектов ASP с подготовленными предварительными настройками 

## Основные элементы
- Подлкючен swagger и scalar
- подготовлена чиста архитектура 
- подключен EF core
- настроен базовый DbContex
- включено использование ControllerBase для API
- Включено использование RazorPage
- настроено использование IOption
